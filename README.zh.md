# WikiGen - AI 代理网页应用

**聊天即工作区，网站生成即产物**

WikiGen 是一个 AI 代理网页应用，其中聊天作为核心工作区，网站生成成为每次对话运行的有形产物。它通过协作式 AI 交互将自然语言讨论转化为功能性网站。

## 概述

WikiGen 重新构想了网站创建方式，将聊天作为与 AI 协作的主要界面。用户不是使用传统设计工具，而是与 AI 代理进行对话来描述、改进和生成网站。每次对话都成为一个创意会话，并产生一个可工作的网站作为输出。

## ✨ 核心功能

### 💬 以聊天为中心的工作区
- **对话式界面**：自然语言聊天作为网站创建的主要工作区
- **上下文保持**：完整的 AI 对话历史，跨会话保持上下文
- **渐进式改进**：网站需求的迭代讨论和改进

### 🚀 AI 驱动的网站生成
- **DeepSeek 集成**：由 DeepSeek AI 模型提供支持的智能网站生成
- **实时生成**：基于对话上下文的实时 HTML、CSS 和 JavaScript 生成
- **响应式输出**：自动生成移动友好的响应式网站

### 👁️ 实时预览与可视化
- **集成预览**：内置网站查看器，实时更新预览
- **多设备模拟**：在桌面、平板和手机屏幕尺寸下预览网站
- **即时反馈**：与 AI 讨论时立即查看更改

### 📁 项目管理
- **组织化项目**：将对话和生成的网站保存为项目
- **可视化仪表板**：通过预览缩略图和元数据浏览项目
- **轻松访问**：快速恢复对话或预览生成的网站

### 🎨 现代用户体验
- **响应式设计**：在所有设备上完全响应的界面
- **深色/浅色主题**：内置主题切换，使用石色调色板
- **基于图标的 UI**：通过 UnoCSS 使用 Phosphor 图标的简洁界面
- **流式响应**：实时 AI 响应流，实现自然交互

## 🏗️ 技术栈

### 前端框架
- **Vue 3.5** 与 Composition API 和 `<script setup>` 语法
- **TypeScript** 用于类型安全和增强的开发体验
- **Vue Router 4** 用于客户端导航

### 样式与 UI
- **UnoCSS** 与 Wind4 预设，用于实用优先的 CSS
- **Preset Attributify** 和 **Transformer Variant Group** 用于简洁的样式
- **Phosphor Icons** 通过 UnoCSS 图标系统集成

### 构建与开发
- **Vite 8** 用于快速开发和优化的生产构建
- **ESLint** 与 `@antfu/eslint-config` 用于代码质量
- **PNPM** 用于高效的包管理

### AI 集成
- **DeepSeek API** 用于 AI 驱动的生成
- **结构化提示**：在专用 TypeScript 文件中存储的提示
- **MarkStream Vue** 用于 Markdown 渲染和流式支持

## 🚀 快速开始

### 先决条件
- 安装 Node.js 18+ 和 PNPM
- DeepSeek API 密钥（用于 AI 功能）

### 安装
```bash
# 克隆仓库
git clone https://github.com/liangmiQwQ/WikiGen.git
cd WikiGen

# 安装依赖
pnpm install

# 配置环境
cp .env.example .env
# 将您的 DeepSeek API 密钥添加到 .env

# 启动开发服务器
pnpm dev
```

### 开发命令
```bash
# 启动开发服务器
pnpm dev

# 构建生产版本
pnpm build

# 类型检查
pnpm typecheck

# 代码检查和修复
pnpm lint:fix

# 预览生产构建
pnpm preview
```

## 📖 使用指南

### 1. 开始对话
首先与 AI 聊天，描述您想要创建的网站。描述您的愿景、需求和偏好。

### 2. 与 AI 协作
进行来回讨论以完善您的想法。AI 会提出澄清问题并给出建议。

### 3. 生成网站
需求明确后，AI 会为您的网站生成 HTML、CSS 和 JavaScript 代码。

### 4. 预览与迭代
在集成预览器中查看生成的网站。向 AI 提供反馈以进行进一步改进。

### 5. 保存为项目
将成功的网站保存为项目，以供将来参考或进行额外修改。

## 🏗️ 架构

### 核心组件
- **聊天界面**：具有消息流和历史的实时聊天
- **网站查看器**：基于 Iframe 的预览，具有响应式缩放
- **项目仪表板**：所有已创建网站的可视化管理
- **AI 集成层**：处理与 DeepSeek API 的通信

### 状态管理
- **组合函数**：用于聊天、项目和 AI 状态的 Vue 组合函数
- **本地存储**：对话和项目的持久化存储
- **响应式更新**：使用 Vue 的响应式系统进行实时 UI 更新

### 提示管理
- **专用提示文件**：AI 提示存储在 `src/prompts/` 目录中
- **类型安全提示**：基于 TypeScript 的提示定义
- **结构化输出**：一致的 AI 响应格式

## 🎯 设计理念

### 聊天作为主要界面
我们相信与 AI 创作的最自然方式是通过对话。聊天提供：
- **上下文理解**：AI 保持对话上下文
- **迭代改进**：自然的来回讨论以完善细节
- **创意协作**：人类和 AI 作为合作伙伴共同工作

### 渐进式披露
功能按需显示：
1. **简单开始**：基本的聊天界面
2. **渐进生成**：网站通过对话逐渐显现
3. **增加复杂性**：随着需求增长添加高级功能

### 用户赋权
- **无需编码**：仅通过对话创建网站
- **完全控制**：用户指导创作过程
- **透明过程**：准确查看对话如何导致输出

## 🔧 配置

### 环境变量
```env
VITE_DEEPSEEK_API_KEY=your_api_key_here
VITE_DEEPSEEK_BASE_URL=https://api.deepseek.com
```

### AI 模型设置
- 模型：DeepSeek 最新模型
- 温度：可配置创意与一致性
- 最大令牌数：可调整响应长度

## 📱 响应式设计

WikiGen 采用移动优先的响应式设计构建：
- **移动优化**：小屏幕上的触摸友好界面
- **自适应布局**：UI 元素根据屏幕尺寸重新排列
- **一致体验**：所有设备上相同的功能

## 🚧 开发指南

### 代码风格
- 使用 Vue 3 Composition API 与 `<script setup>`
- 遵循 TypeScript 最佳实践
- 使用带有 attributify 语法的 UnoCSS 实用类
- 将 AI 提示存储在专用 TypeScript 文件中

### UI/UX 标准
- 通过 `i-ph-*` 类使用 Phosphor 图标
- 为所有组件实现响应式设计
- 遵循石色调色板指南
- 避免使用表情符号；改用图标

### 质量保证
- 更改后运行 `pnpm run lint:fix`
- 确保 `pnpm run typecheck` 通过
- 测试跨断点的响应行为
